Future Lebanon

A comprehensive web platform designed to help Lebanese students navigate higher education and explore career paths. Developed as a showcase project during my training with CodeBrave, it features an interactive directory of 160+ university majors, university insights, and a personalized "Passion Matcher" tool. The platform demonstrates how modern front-end development and AI-assisted workflows can be combined to create engaging, user-centered web applications.

Cosmic Capsule

A futuristic web application designed to securely preserve personal memories, letters, and photos behind custom time-lock logic until a chosen future date. Inspired by a school activity where I realized handwritten letters could easily be lost over time, I created this platform as a permanent digital space that is always with you. Built with HTML, CSS, JavaScript, and LocalStorage for persistent data storage, the application features automatic background saving, an interactive statistics dashboard, light and dark theme customization, full responsiveness, and immersive audio feedback that plays a magical sound effect when a time capsule is successfully locked.
